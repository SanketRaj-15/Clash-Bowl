# 🎮 Clash Bowl - Unity Mini Game

A fast-paced mobile mini-game built using Unity where the player controls a ball to collect pickups within a limited time.

---

## 📱 Gameplay Preview

* Control a red ball inside a bounded arena
* Collect glowing pickups to increase score
* Beat the timer and achieve the best score

---

## ✨ Features

* 🎯 Smooth player movement (mobile tilt controls)
* ⏱️ Timer-based gameplay
* 🟡 Dynamic pickup system with visual effects
* 🔊 Sound effects for pickups and win condition
* 🏆 Score + Best Time tracking
* 🎨 Clean minimal UI with TextMeshPro

---

## 🛠️ Tech Stack

* Unity 2022 (3D)
* C#
* TextMeshPro
* Rigidbody Physics System

---

## 📂 Project Structure

* `Assets/` → Game assets, scripts, prefabs
* `ProjectSettings/` → Unity configuration
* `Packages/` → Dependencies

---

## 🎮 Controls

* Mobile Tilt → Move player
* Objective → Collect all pickups before time ends

---

## 🚀 How to Run

1. Clone the repository
2. Open in Unity Hub
3. Click Play
